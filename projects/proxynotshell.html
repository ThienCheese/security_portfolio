Defensive recommendations

What to hunt or harden

Registry & Tasks

Persistence monitoring

Monitor Run keys and scheduled task creation, especially paths under user-writable or uncommon directories.

Memory

LSASS and private executable regions

Alert on suspicious LSASS access and memory regions that suggest injected code.

AD

DCSync indicators

Hunt Event ID 4662 with replication-related rights and unexpected domain replication activity.

Exchange

Patch and log review

Patch Exchange, inspect IIS/Exchange logs, and monitor Remote PowerShell patterns.

Control

AppLocker / WDAC

Block unapproved binaries and script execution paths to reduce post-exploitation tooling.

IR

Kerberos recovery

If krbtgt exposure is suspected, plan credential reset and Kerberos trust recovery carefully.
