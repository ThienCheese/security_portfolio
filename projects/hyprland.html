Fedora Hyprland Desktop Customization

A personal Linux customization project showing my interest in building a fast, polished, and automated daily-driver environment.

View screenshots Watch demo video Dotfiles GitHub
Fedora Linux Hyprland Waybar SwayNC Fish Shell

What I built

  • A Fedora-based Wayland desktop using Hyprland as the compositor.
  • Dynamic color theming where the wallpaper palette propagates across the bar, notification center, and terminal workflow.
  • A GNOME-like control center and notification workflow using SwayNC.
  • A music workflow using MPD/rmpc and Fish shell automation for local music management.
  • A screen-sharing friendly Wayland setup using PipeWire and xdg-desktop-portal-hyprland.

System components

Stack overview

OS

Fedora Linux as the base system for a modern desktop environment.

Compositor

Hyprland for a fast Wayland workflow with custom window rules and animations.

Status / control

Waybar and SwayNC for system status, notification center, and control widgets.

Theming

Wallust dynamic colors to sync UI palette with the active wallpaper.

Shell

Fish shell for an interactive terminal workflow and small automation scripts.

Audio / media

PipeWire, WirePlumber, MPD, and rmpc for audio and music management.

Screen sharing

xdg-desktop-portal-hyprland configured for Wayland screen capture.

Dotfiles

Repository-based configuration to keep the environment reproducible.
